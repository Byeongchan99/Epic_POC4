# Magic Circle Slash - POC

## 🎯 핵심 경험
**"적 마법사의 주문을 손으로 베어내는 긴장감"**

마법 세계관에서 플레이어는 다른 마법사들이 마법을 시전할 때 나타나는 마력의 형상(마법진)을 볼 수 있습니다.
마우스 드래그로 이 마법진을 **직접 잘라내어** 마법을 취소시키는 독창적인 메카닉입니다.

## 🎮 조작 방법
- **마우스 클릭 + 드래그**: 마법진을 자르기
- **목표**: 3초 안에 **빨간색 약점 선분**을 모두 끊어내기
- **약점**: 빨간색으로 표시된 특정 선분 (전체의 30%)
- 실패 시 마법이 발동되어 패배

## 🌟 POC 특징

### ✂️ 독창적인 메카닉
- **약점 시스템**: 빨간색 약점만 정확히 파괴하는 전략적 플레이
- **선분 교차 판정**: 실시간으로 드래그 경로와 마법진 선분 간 교차 계산
- **물리적 감각**: 마우스로 "자르는" 행위 자체가 게임플레이
- **시각적 피드백**: 끊어지는 순간 파티클 효과 + 카메라 쉐이크

### 📈 12개 스테이지 시스템
- **Stage 1-4**: 기본 도형 (Circle, Triangle, Square, Pentagram)
- **Stage 5-8**: 복잡한 별 (Hexagram, Heptagram, Octagram, DoublePentagram)
- **Stage 9-12**: 고난이도 패턴 (CrossPattern, Spiral, InfinitySymbol, ComplexRune)
- **진행 방식**: 클리어 시 다음 스테이지, 실패 시 현재 스테이지 재시작
- **최종 목표**: 12스테이지 완전 클리어

### 🎨 시각적 요소
- 마법진이 **천천히 그려지는** 애니메이션
- 자를 때 나타나는 **발광 파티클 효과**
- 빨간색 검기 **트레일 렌더링**
- 타이머 긴장감

### 주요 컴포넌트
- **MagicCircle.cs**: 마법진 패턴 생성 및 그리기
- **SlashDetector.cs**: 마우스 드래그 감지 및 교차 판정
- **LineSegment.cs**: 선분 데이터 구조 및 교차 알고리즘
- **GameManager.cs**: 게임 로직 및 스테이지 관리
- **CutEffect.cs**: 마법진 절단 시 파티클 효과

## 🚀 실행 방법

### 초기 세팅 (최초 1회만)
1. Unity에서 프로젝트 열기 (Unity 2021.3 이상 권장)
2. **[SETUP_GUIDE.md](SETUP_GUIDE.md) 참고하여 씬 세팅**
   - GameManager, SlashDetector, UI 생성
   - MagicCircle Prefab 생성
   - Inspector에서 값 연결

### 플레이
1. `Scenes/SampleScene.unity` 열기
2. **Play 버튼 클릭**
3. 마우스 드래그로 마법진 자르기!

> **수동 세팅 방식**: Inspector에서 모든 값을 자유롭게 조정할 수 있습니다.

## 📊 검증 포인트

### ✅ 즉시 검증 가능한 질문들
1. **재미**: 30초 플레이로 "다시 하고 싶다" 느껴지는가?
2. **명확성**: 플레이어가 조작법을 바로 이해하는가?
3. **만족감**: 마법진을 끊을 때 쾌감이 있는가?
4. **긴장감**: 타이머와 복잡해지는 패턴이 압박을 주는가?
5. **독창성**: "이 게임 어디서 본 적 있어?" → "아니, 처음이야!"

### ❌ POC가 아닌 것

- 스토리나 세계관 깊이

## 🔮 확장 가능성

이 POC가 성공적이라면:
- 여러 마법진 동시 시전 (멀티태스킹)
- 회전하는 마법진 (난이도 증가)
- 특정 순서로 끊어야 하는 퍼즐 요소
- 마법진을 끊어 얻은 마력을 자신의 공격에 사용
- PvP: 한 명은 마법 시전, 한 명은 방해
