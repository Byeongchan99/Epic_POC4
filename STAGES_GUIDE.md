# 12 스테이지 시스템

## 🎮 스테이지 구성

Magic Circle Slash는 **12개의 스테이지**로 구성되어 있으며, 각 스테이지마다 다른 마법진 패턴이 등장합니다.

| 스테이지 | 패턴 | 난이도 | 설명 |
|---------|------|--------|------|
| 1 | Circle | ⭐ | 단순한 원형 - 튜토리얼 |
| 2 | Triangle | ⭐ | 정삼각형 |
| 3 | Square | ⭐⭐ | 정사각형 |

## 🎯 게임 진행

### 클리어 조건
- 각 스테이지의 **모든 약점(빨간색 선분)을 파괴**하면 클리어
- 3초 시간 제한 내에 완료해야 함

### 성공 시
- **다음 스테이지로 진행**
- 남은 시간 × 100점 획득
- 총점에 누적
- 2초 후 자동으로 다음 스테이지 시작

### 실패 시
- **현재 스테이지 재시작**
- 획득한 점수는 유지
- 2초 후 같은 스테이지 다시 시작
- 처음부터 시작하지 않음!

### 전체 클리어
- 12스테이지를 모두 클리어하면 게임 완료
- 최종 점수 표시
- 축하 메시지
- 5초 후 처음부터 다시 시작

## 📊 점수 시스템

### 점수 계산
```
스테이지 점수 = 남은 시간(초) × 100
```

**예시:**
- 3초 남기고 클리어: 300점
- 1.5초 남기고 클리어: 150점
- 0.1초 남기고 클리어: 10점

### 총점 누적
- 각 스테이지의 점수가 누적됨
- 실패해도 점수는 유지 (다시 도전 가능)
- 최고 점수: 약 3600점 (각 스테이지 3초 만점 기준)

## 💡 난이도 진행

### 초반 (Stage 1-4)
- 단순한 도형
- 약점 1~2개
- 익히기 쉬움

### 중반 (Stage 5-8)
- 복잡한 별 모양
- 약점 3~4개
- 전략적 플레이 필요

### 후반 (Stage 9-12)
- 매우 복잡한 패턴
- 약점 5~10개
- 고난이도 챌린지

## 🔧 난이도 조절

### 쉽게 만들기
**GameManager** Inspector:
- **Cast Time**: `5초` (더 많은 시간)

**MagicCircle Prefab** Inspector:
- **Weakpoint Ratio**: `0.1` (약점 10%)

### 어렵게 만들기
**GameManager** Inspector:
- **Cast Time**: `2초` (적은 시간)

**MagicCircle Prefab** Inspector:
- **Weakpoint Ratio**: `0.4` (약점 40%)

## 🎮 플레이 팁

### 스테이지 1-4 (연습)
- 천천히 정확하게
- 약점 위치 파악
- 조작에 익숙해지기

### 스테이지 5-8 (전략)
- 가까운 약점부터
- 빠른 손놀림
- 시간 관리

### 스테이지 9-12 (도전)
- 약점 분포 빠르게 파악
- 최단 경로로 자르기
- 손 떨림 주의

## 🏆 도전 과제

## 📝 개발자 노트

### 스테이지 추가 방법
`GameManager.cs`의 `GetPatternForStage` 메서드에 패턴 추가:
```csharp
case 13: return MagicCircle.CirclePattern.MyNewPattern;
```

그리고 `TOTAL_STAGES` 상수를 변경:
```csharp
private const int TOTAL_STAGES = 13;
```

### 스테이지 순서 변경
`GetPatternForStage`에서 case 순서 변경

### 특정 스테이지만 테스트
`Start()` 메서드 수정:
```csharp
void Start()
{
    currentStage = 4; // 5스테이지부터 시작
    StartNextStage();
}
```
